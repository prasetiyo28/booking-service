# Booking API Services

an api booking with laravel version 11.x

## Requirement
- PHP ^8.1
- Composer
- MySQL

## Installation

- install the dependency

```bash
composer install
```

- migrate the database
```bash
PHP artisan migrate
```
- running the project
```bash
PHP artisan serve
```
